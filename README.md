# Sistema de Tickets — Unidad 3

Mini aplicación web hecha con **HTML, Bootstrap y JavaScript** para la práctica de la Unidad 3.

## ✔ Funcionalidades

- Crear tickets con: Nombre, Email, Mensaje, Prioridad.
- Validaciones en JavaScript.
- Objeto `Message` con métodos.
- Lista dinámica (DOM).
- Marcar como leído.
- Eliminar ticket.
- Contador de tickets urgentes.
- Persistencia con localStorage.
- Responsivo con Bootstrap.
